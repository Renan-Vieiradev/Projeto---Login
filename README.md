# Projeto - Login
Este é um projeto básico de tela de login desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, incluindo estruturação com HTML, estilização com CSS e validação com JavaScript.

Funcionalidades:
-Campo de entrada para e-mail e senha;
-Validação de formulário com JavaScript;
-Feedback visual para o usuário (erros de preenchimento);
-Interface simples e responsiva.

Por enquanto, estou me dedicando e estudando mais a fundo backend, assim que possível vou atualizar e fazer deste projeto um sistema completo.
